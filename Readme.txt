()-event binding syntax is used to passed the 
data to HTML to ts.
[]-a property binding is used to passed the data
from ts to HTML.
[()]- this is called as banana in box syntax 
which is used for two way data binding.
But in order to use two way data binding 
we have to use built-in module known as FormsModule.
Two way data binding is very important feature for angular 
and very popular.

All form submission is typically done using 
two way data binding syntax.  
[(ngModel)]="username" 
where ngModel is built-in attribute given by angular.
username is field define in the ts field .
